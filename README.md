# بناء للمقاولات والديكور (Benaa)

الموقع الرسمي لشركة بناء للمقاولات والديكور - المرحلة الأولى (الواجهة الأمامية).

## المميزات
- تصميم عصري واحترافي يدعم اللغة العربية (RTL).
- متجاوب تماماً مع جميع الشاشات (Mobile/Tablet/Desktop).
- أزرار تواصل سريعة (واتساب، ماسنجر، اتصال مباشر).
- متجر للعقارات والأراضي مدمج في الصفحة.
- شاشة تحميل ذكية وسرعة في الأداء.

## التقنيات المستخدمة
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Vite (Build Tool)

## كيفية التشغيل محلياً
```bash
git clone [repository-url]
cd [folder-name]
npm install
npm run dev
```

أو عبر Python:
```bash
python3 -m http.server 3000
```
